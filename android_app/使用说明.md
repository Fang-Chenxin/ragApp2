# Android Studio 超详细新手使用教程

## 第一步：安装 Android Studio
1. 官网下载地址：https://developer.android.com/studio
2. 一路默认下一步安装，记得勾选 Android SDK 组件
3. 首次启动等待它自动下载完成所有SDK和依赖

## 第二步：打开我们准备好的项目
1. 启动 Android Studio
2. 选择 "Open an Existing Project"
3. 在文件浏览器中选中我们的 `android_app` 文件夹
4. 点击 OK，等待几分钟，Android Studio会自动同步所有Gradle依赖

## 第三步：创建模拟器（或者连接自己的安卓真机）
方式A - 安卓模拟器（推荐新手）：
- 点击顶部菜单栏的 Device Manager 图标（手机样子）
- 点击 Create Device
- 选一个像素5之类的手机型号，下一步
- 选Android 13.0系统镜像，下载完成后选它，完成创建
- 点击播放按钮启动模拟器

方式B - 用自己真实的安卓手机：
- 手机上开启「开发者选项」和「USB调试」
- 用USB线把手机连到电脑上，Android Studio会自动识别到你的设备

## 第四步：点击运行按钮！
- 顶部工具栏选好你刚才创建的模拟器/真机
- 点击那个绿色的三角形「Run」按钮
- 等待几十秒，App就自动安装并启动在你的设备上啦！

## 第五步：后端服务和App联调
- 先运行我们的Python后端服务 `backend/main.py`，端口8000
- 如果你用模拟器：不用改代码！默认的10.0.2.2就是模拟器访问宿主机的特殊地址，直接就能连通
- 如果你用真机：请把 MainActivity.kt 里的 BACKEND_URL 改成你电脑在局域网里的IP地址，比如 http://192.168.x.x:8000/api/chat
