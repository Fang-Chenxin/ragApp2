# Android Studio 国内全场景提速配置

## 1. Gradle Wrapper 下载加速（最重要！）

打开 Android Studio 设置 → Build, Execution, Deployment → Build Tools → Gradle

找到 `Gradle user home` 目录，通常是 `~/.gradle` (Linux/Mac) 或 `C:\Users\你的用户名\.gradle` (Windows)

在这个 .gradle 文件夹下面新建一个叫 `init.gradle` 的文件，粘贴下面全部内容：
```groovy
allprojects {
    repositories {
        def ALIYUN_REPO = "https://maven.aliyun.com/repository"
        all { ArtifactRepository repo ->
            if(repo instanceof MavenArtifactRepository){
                def url = repo.url.toString()
                if (url.startsWith('https://repo1.maven.org/maven2') || url.startsWith('https://repo.maven.apache.org/maven2')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $ALIYUN_REPO/public"
                    remove repo
                }
                if (url.startsWith('https://dl.google.com/dl/android/maven2/')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $ALIYUN_REPO/google"
                    remove repo
                }
            }
        }
        maven { url "${ALIYUN_REPO}/public" }
        maven { url "${ALIYUN_REPO}/google" }
        maven { url "${ALIYUN_REPO}/central" }
        maven { url "${ALIYUN_REPO}/gradle-plugin" }
    }
}
```

## 2. 项目里我们已经自动配置好阿里云镜像源了
settings.gradle.kts 里所有依赖下载已经默认走阿里云镜像，不用再改任何东西！

## 3. Android SDK 下载加速
在 Android Studio 设置里找 System Settings → Android SDK，下载SDK如果慢的话，在SDK Manager的右上角勾选「Enable HTTP Proxy」，镜像地址填国内镜像站即可。
